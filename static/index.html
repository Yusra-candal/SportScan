🏃 Spor Karne

Öğrenci Atletik Performans Yönetim Sistemi

✅ Sistem Aktif

Spor Karne servisi başarıyla çalışıyor. Öğrenci yönetimi, performans değerlendirme ve rapor kartı özelliklerine erişebilirsiniz.

📋 Ana Özellikler

  • Öğrenci bilgisi yönetimi (ad, sınıf, doğum tarihi, boy, kilo)
  • Spor performansı değerlendirmesi (voleybol, basketbol, futbol)
  • Dijital rapor kartı ve istatistikler
  • Sıçrama yüksekliği analizi (video analiz)
  • Spor bazında sıralama ve karşılaştırma

🔧 API Endpoints

  • GET /api/healthz - API durumu
  • GET /api/students - Öğrenci listesi
  • POST /api/metrics - Performans metriği ekle
  • POST /video-api/analyze - Sıçrama analizi
